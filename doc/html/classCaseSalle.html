Detailed Description

class des cases qui composent les salles dans le jeu. Une Salle de base est composée d'un tableau de CaseSalle de taille variable. Il est à noter que les CaseSalle servent uniquement à la gestion et la génération de décors et d'ennemis, et que les mouvements d'entités ne dépendent pas de ces CaseSalle.
