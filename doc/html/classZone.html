Detailed Description

Gère l'instance où crapahute Perso. Une seule Zone sera en général générée à chaque instant. Elle contient des informations sur toutes les Salle de l'instance (rangées dans un tableau 11*11) et instancie toutes les Salle où Perso évoluera.
